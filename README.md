# HTML to Markdown converter

Web page grabber that outputs markdown

## Demo files

* **ref.md** - unofficial k reference
* **shakti.md** - official k reference
* **ck.md** - http://www.nsl.com/papers/ck.htm

## How to run

```
npm i
node app.js <url> <root selector>
```
